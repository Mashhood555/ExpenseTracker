# ExpenseTracker 💰

Your first iOS app! A personal expense tracker built with SwiftUI and SwiftData.

## Features

✨ **Track Expenses**
- Add, edit, and delete expenses
- Organize by category (Food, Transport, Entertainment, Shopping, Bills, Other)
- Add notes and dates to each expense

📊 **Visualize Spending**
- Beautiful bar charts showing category breakdown
- Weekly, monthly, and yearly summaries
- Progress bars with percentage breakdowns

🎨 **Modern Design**
- Clean SwiftUI interface
- Gradient backgrounds
- Smooth animations
- Floating action button

## Tech Stack

- **SwiftUI** - Apple's modern UI framework
- **SwiftData** - Local data persistence (iOS 17+)
- **Swift Charts** - Data visualization
- **Swift 5.9+** - Latest Swift features

## How to Use

1. **Add an Expense**: Tap the blue + button
2. **Filter by Category**: Tap category chips at the top
3. **View Summary**: Switch to the Summary tab at the bottom
4. **Delete**: Swipe left on any expense
